*READ ME*
1.The file that contains the code is named MCL.py.
2.The required dataset is passed as a parameter on line 9 (.txt file).
3.The expansion and inflation parameters are initialized on lines 13 and 14 respectively.
4.The required .net file is passed as a parameter on line 104.
5.The .clu file is created on our systems as an empty file.
6.It is then opened for writing on line 108.
7.Running the code after making the above changes should give the .clu file in the required format.
8.This .clu file is then passed on to Pajek for Visualisation.
